## 📊 功能列表

### Excel 处理
| 功能 | 说明 | 适用场景 |
|------|------|----------|
| 🔀 **合并表格** | 将多个 Excel/CSV 合并为一个 | 汇总多个月/季度数据 |
| 📂 **拆分表格** | 按列值拆成多个文件 | 按部门/地区分发数据 |
| 🧹 **清洗数据** | 去空行、空格、重复项 | 清理导入前的脏数据 |
| 🔍 **搜索筛选** | 按关键词过滤行 | 快速找到目标客户 |
| ⚖️ **对比差异** | 找出两个表格的区别 | 对账、版本对比 |
| 🗂️ **去重客户** | 按姓名/手机号去重 | 客户名单去重 |

### 图片处理
| 功能 | 说明 |
|------|------|
| 📐 **调整尺寸** | 批量缩放到指定像素 |
| 🗜️ **压缩图片** | 减小体积保持清晰度 |
| 💧 **添加水印** | 文字或图片水印 |
| 🔄 **格式转换** | PNG ↔ JPG ↔ WebP |

### 📂 文件处理
- **批量重命名** — 支持 {原名}_{编号}_{日期} 等模板
- **PDF 转图片** — 将 PDF 页面转为图片

### 🛠️ 开发工具
- **JSON 格式化** — 美化、压缩、校验
- **Base64 编解码** — 图片与 Base64 互转
- **Markdown 预览** — 实时预览 Markdown
- **颜色转换** — HEX / RGB / HSL 互转
- **UUID 生成器** — 批量生成 UUID

### 🔧 实用工具
- **密码生成器** — 自定义长度和字符类型
- **字数统计** — 中英文字符、单词统计
- **二维码生成** — 自定义尺寸和颜色
- **IP 查询** — 查看本机 IP 地址
- **繁简转换** — 繁体与简体互转
