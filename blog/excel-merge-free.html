方法二：Excel自带的VBA宏

如果你熟悉Excel，可以用VBA宏来合并。按 Alt+F11 打开VBA编辑器，插入模块，粘贴以下代码：

Sub MergeFiles()
    Dim folder As String, file As String
    Dim lastRow As Long, i As Integer
    folder = "C:\你的文件夹路径\"
    file = Dir(folder & "*.xlsx")
    i = 0
    Do While file <> ""
        If file <> ThisWorkbook.Name Then
            Workbooks.Open folder & file
            Sheets(1).UsedRange.Copy
            ThisWorkbook.Sheets(1).Cells(Rows.Count, 1).End(xlUp).Offset(1, 0).PasteSpecial xlPasteValues
            Workbooks(file).Close False
            i = i + 1
        End If
        file = Dir
    Loop
    MsgBox "已合并 " & i & " 个文件"
End Sub

缺点： 需要开启宏，部分公司禁止使用VBA。

方法三：Power Query（Excel 2016+）

  1. 打开Excel → 数据 → 获取数据 → 从文件 → 从文件夹
  2. 选择包含所有Excel文件的文件夹
  3. 点击「合并和转换数据」
  4. 选择要合并的工作表
  5. 点击「关闭并上载」

优点： Office原生功能，无需额外工具
缺点： 仅支持Excel 2016及以上版本

常见问题

Q: 合并后数据错位怎么办？

确保所有文件的表头（第一行）格式一致。在线工具会自动识别表头，只合并数据行。

Q: 文件太大合并失败？

在线工具有文件大小限制。超大文件建议用Power Query或VBA方法。

Q: CSV文件能合并吗？

可以，支持 .xlsx、.xls、.csv 三种格式混合合并。
